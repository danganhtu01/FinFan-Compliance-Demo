# FinFan Compliance — public demo

Static demo of the FinFan AML screening engine (Rust → WebAssembly). Built from
a private repository; this repo holds only the compiled site. Screening runs
entirely in the browser over cleaned **public** sanctions-authority data —
no customer data, no internal lists, and typed names never leave the visitor's
browser. Pre-production demo; the full platform runs on the FinFan intranet.
